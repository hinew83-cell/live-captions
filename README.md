# LiveCaptions (실시간 PC 오디오 자막기)

LiveCaptions는 PC 내부에서 발생하는 소리(유튜브 영상, 음악, 강의 음성 등)를 별도의 장비나 가상 케이블 없이 실시간으로 받아적어 주는 **Windows 특화 실시간 오디오 자막 변환 및 번역 애플리케이션**입니다.

## ✨ 주요 기능
1. **Windows WASAPI 루프백 수신**: 스테레오 믹스 설정 없이 스피커나 헤드폰(예: AirPods 등 블루투스 기기)으로 나가는 소리를 실시간으로 자동 캡처합니다.
2. **실시간 자막 피드 & 오디오 다시듣기**: 자막을 한 문장 단위로 밀도 있게 출력하며, 각 자막 옆의 ▶ 재생 버튼을 눌러 자막이 추출되었던 **실제 사운드를 다시 청취**할 수 있습니다.
3. **짧은 노이즈 필터링**: 숨소리, 키보드 타건음, 마우스 클릭음 등의 0.8초 미만의 무의미한 노이즈를 스스로 필터링하여 미인식 구간(`!`) 표시를 최소화합니다.
4. **글로벌 다국어 및 실시간 번역**: 영어(`en-US`), 일본어(`ja-JP`) 등 선택한 언어로 받아적으며, 외국어의 경우 **한국어 번역본을 괄호 안에 동시 노출**합니다.
5. **텍스트 기록 히스토리 (좌우 6:4 레이아웃)**: 왼쪽에서는 재생 제어가 가능한 자막 피드가 표시되고, 오른쪽에서는 복사 및 저장이 편리하도록 기호나 미인식 표시가 없는 **깨끗한 텍스트 기록 히스토리**가 표시됩니다.
6. **양방향 실시간 텍스트 수정**: 화면의 자막이나 히스토리를 직접 더블클릭/클릭하여 타이핑하면, 두 패널의 텍스트와 내보내기 버퍼가 실시간으로 연동되어 즉석 수정됩니다.
7. **Apple Siri 스타일 파형 시각화**: 소리 감지 유무 및 크기에 맞춰 Siri/Soundwave 스타일의 부드러운 오디오 파형 그래픽이 캔버스에 렌더링됩니다.
8. **플로팅 오버레이 모드**: 화면 위에 작게 띄워두고 투명도를 유지한 채 마우스로 끌어서 어느 곳에든 배치할 수 있습니다.

## 🛠️ 준비 및 설치 방법

### 1. 요구사항
- Windows OS (WASAPI loopback 이용을 위해 필수)
- Python 3.8 이상

### 2. 가상환경 세팅 및 의존성 패키지 설치
프로젝트 루트 폴더에서 터미널을 열고 아래 명령어를 순서대로 실행합니다:

```bash
# 1. 가상환경 생성
python -m venv venv

# 2. 가상환경 활성화 (Windows PowerShell 기준)
.\venv\Scripts\Activate.ps1

# 3. 필수 패키지 설치
pip install fastapi uvicorn speechrecognition pyaudiowpatch numpy
```

## 🚀 실행 방법

가상환경이 활성화된 상태에서 아래 명령어로 서버를 구동합니다:

```bash
python server.py
```

서버가 구동되면 웹 브라우저를 열고 다음 주소로 접속합니다:
👉 **[http://localhost:8000](http://localhost:8000)**

## 📂 프로젝트 구조
- `server.py`: FastAPI 백엔드로 오디오 스트림 캡처, 무음 감지 분할, STT 및 번역 API 처리를 담당합니다.
- `static/`: 프론트엔드 정적 파일 폴더입니다.
  - `index.html`: 대시보드 레이아웃 마크업.
  - `index.css`: 글래스모피즘 테마 및 와이드 좌우 분할 CSS.
  - `index.js`: 웹소켓 연결, 오디오 재생 제어, 캔버스 그래픽 및 로컬 편집 동기화 JS.
- `.gitignore`: 불필요한 가상환경(`venv`) 및 임시 오디오 캐시 파일이 깃에 올라가지 않도록 제어합니다.
